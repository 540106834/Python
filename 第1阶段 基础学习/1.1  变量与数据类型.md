
### 🧠 1. 变量与数据类型

#### 🎯 SRE 常见用途

* 存储告警信息（字符串）
* 记录监控结果（字典）
* 计算资源使用率（数值）
* 批量检测多个目标（列表）

#### 📘 示例讲解：

```python
# CPU、内存使用率（假设从监控系统获取）
cpu_usage = 72.5
memory_usage = 83.1

# 服务状态（True 表示正常）
is_service_up = True

# 服务列表
services = ["auth", "payment", "monitoring"]

# 告警阈值（字典）
threshold = {
    "cpu": 80,
    "memory": 85
}

print(f"当前CPU: {cpu_usage}%，内存: {memory_usage}%")

if cpu_usage > threshold["cpu"]:
    print("⚠️ CPU 使用率超过阈值！")
else:
    print("✅ CPU 正常")

print(f"服务数量: {len(services)}")
```

#### 💡 输出结果示例：

```
当前CPU: 72.5%，内存: 83.1%
✅ CPU 正常
服务数量: 3
```

## 🧩 练习题：CPU 使用率监控脚本

**题目回顾：**
假设三个服务器的 CPU 使用率数据为 `[72, 88, 65]`，阈值是 `80`。
逐个检查：

* 超过阈值打印告警信息
* 否则打印“正常”

---

### ✅ 参考答案

```python
# CPU 使用率列表
cpu_usages = [72, 88, 65]

# 阈值
threshold = 80

# 遍历每个服务器
for i, usage in enumerate(cpu_usages, start=1):
    if usage > threshold:
        print(f"Server{i}: ⚠️ CPU 超过阈值！（当前 {usage}%）")
    else:
        print(f"Server{i}: 正常（当前 {usage}%）")
```

---

### 💡 输出结果

```
Server1: 正常（当前 72%）
Server2: ⚠️ CPU 超过阈值！（当前 88%）
Server3: 正常（当前 65%）
```

---

### 🧠 知识点讲解

| 知识点             | 作用                      |
| --------------- | ----------------------- |
| `enumerate()`   | 同时获取列表元素和索引（这里用来给服务器编号） |
| `if` 判断         | 检查是否超过阈值                |
| `f"..."` 格式化字符串 | 方便输出带变量的文本              |
